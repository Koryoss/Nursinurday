📡 핵심 데이터 흐름 (v3.1 상호관련성 9.0/10)

자동 감지 흐름:
⌚ Gyro+Accel (10Hz) + HR (HKAnchored) + Voice (dB)  →  WatchDizzinessDetector
    → 에피소드 감지 (confidence ≥ 60%)  →  📱 WatchConnectivity  →  WatchConnectivityManager
    → LocationTracker.recordEpisodeAtCurrentLocation()  +  CareFlowAPIClient
    → 🖥 POST /api/sensors/dizziness  →  NANDA 분류  →  🌐 Dashboard 표시

HealthKit 동기화 흐름:
⌚ HealthKit 안정 HR  →  📱 HealthKitManager.onRestingHRUpdated
    → WatchConnectivityManager.sendBaselineHRToWatch()  →  ⌚ Detector 개인화
    → HealthKitManager.onDailyDataReady  →  🖥 POST /api/sensors/healthkit

AI 챗봇 흐름:
🌐 사용자 채팅 입력  →  🖥 /api/chat (센서 컨텍스트 주입)  →  🤖 AI API  →  심리 지원 응답
