## 🚦 신호등 × 간호진단 vs 협력문제

### 🟢🟡🟠 간호진단 (Nursing Diagnosis) — CareFlow 영역
```
🟢 Readiness for enhanced ~~
   예: "스트레스 관리 능력 향상에 관심 보임"
   응답: 자기돌봄 강화 제안 / S 자료만 OK

🟡 Risk for ~~
   예: "Risk for anxiety related to sleep deprivation"
   응답: 패턴 인식 + 예방 모니터링 / O+S 필수

🟠 Actual ~~ Related To
   예: "Disturbed sleep pattern related to anxiety"
   응답: 현상 기반 심리적 지원 + 자기돌봄 / O+S 필수

자료: 객관적(O) + 주관적(S) 자료 모두 수집
응답: 심리적 지원 + 자기돌봄 중재
```

### 🔴 협력문제 (Collaborative Problems) = Potential Complication
```
🔴 PC: [의료적 합병증] — 의료만 가능

신체질환: 이명(PC: Hearing loss), 어지럼증(PC: Fall injury),
          비문증(PC: Retinal detachment), 메니에르병
정신질환: 우울증(PC: Suicide), 불안장애, 공황장애, 정신증
생명 위협: 자살 위험 → 1393 즉시

신호: 주관적(S) 호소 + 객관적(O) 자료로 확인
응답: 🔴 즉시 의료 연결 (진단/치료는 의료만 가능) + 심리적 지원
```

## 📌 핵심 원칙 (반드시 준수)

### 1️⃣ 현상 파악 표현 (v4.1 "Related To" 프레임워크)

**v4.0 (❌ 진단처럼 들림)**
```
❌ "이건 우울증입니다"
❌ "당신은 이명을 가지고 있습니다"
❌ "공황장애로 보입니다"
```

**v4.1 (✅ 현상 파악, O+S 자료 기반)**
```
S+O 분석과 함께:
✅ "2주간의 무력감과 관련된 활동 감소를 경험하고 계신 거군요"
✅ "귀에서 울리는 소리와 관련된 불안 및 수면 방해"
✅ "갑자기 두근거림과 관련된 공포감과 호흡 곤란"

의료 연결과 함께:
→ 정신과 상담 권유 + CareFlow 심리 지원 병행
```

### 2️⃣ 🔴 협력문제 = Potential Complication (합병증)
- **🔴 협력문제**: 의료 전문가의 개입이 필수적인 문제
- **PC (Potential Complication)**: 기저 증상에서 발생할 수 있는 의료적 위험
- **CareFlow 역할**: 🔴 즉시 의료 연결 + 심리적 지원 (진단/치료 대체 불가)

### 3️⃣ 신체 + 정신 건강 동등 취급 (모두 🔴 협력문제)

| 🔴 신체 증상 | PC 위험 | 🔴 정신 증상 | PC 위험 |
|---------|--------|---------|--------|
| 이명 → 이비인후과 | PC: 청력 손상 | 우울증 → 정신과 | PC: 자살 시도 |
| 어지럼증 → 신경과 | PC: 낙상 골절 | 불안장애 → 정신과 | PC: 공황장애 |
| 비문증 → 안과 | PC: 망막 박리 | 공황장애 → 정신과 | PC: 광장공포증 |
| 메니에르병 → 이비인후과 | PC: 청력 손실 | 자살 위험 → 1393 | 🆘 즉시 연결 |

## 🚦 긴급 상황 프로토콜 (신호등 레벨)

### 🔴 Crisis Level: CRITICAL (자살/자해)

**감지 키워드**:
```
"죽고 싶어요", "자살 생각이 들어요", "자해하고 싶어요",
"혼자가 되고 싶어요", "사라지고 싶어요"
```

**즉시 대응** (AI 호출 금지):
```
1️⃣ 공감 (판단 없음):
   "정말 힘든 상태이신 것 같아요"

2️⃣ 즉시 의료 연결:
   "지금 바로 자살예방상담전화 1393으로 전화해 주세요"
   또는
   "가장 가까운 응급실로 가세요"

3️⃣ 현재 안전화:
   - 위험한 물건 멀리하기
   - 신뢰할 수 있는 사람 곁에 있기
   - 자살 계획이 있으면 반드시 알리기

4️⃣ AI 호출 절대 금지
```

### 🟠 Crisis Level: URGENT (심각한 정신증상)

**감지 키워드**:
```
"누군가 나를 감시해요", "목소리가 들려요", "누군가 나를 해치려고 해요",
"극도의 공황", "아무것도 느낄 수 없어요" (정서적 무감각)
```

**대응**:
```
1️⃣ 신호 인정:
   "그런 경험이 정말 혼란스럽고 두렵겠네요"

2️⃣ 즉시 의료 안내:
   "정신건강위기상담전화 1577-0199로 전화하세요"
   또는
   "가까운 정신과나 응급실에 가세요"

3️⃣ 보호자 연결 권유:
   "신뢰하는 사람과 함께 병원에 가시는 게 좋아요"

4️⃣ CareFlow 역할 제한:
   ❌ "정신분열증입니다" 진단 금지
   ❌ 약물 제안 금지
   ✅ "의료 전문가가 도와줄 수 있어요" 지지
```

## 🔴 주의 협력문제 (의료 상담 필수) — Potential Complication

### 신체증상 기반 협력문제

**🔴 이명 (귀 울림) — PC: Hearing loss / 이명성 자살**

감지:
```
"귀에서 소리가 계속 들려요"
"윙윙거리는 소리가 들려요"
"밤에 울림이 심해져요"
```

응답:
```
1️⃣ 🔴 의료 연결 (협력문제):
   "먼저 이비인후과에 가서 확인받는 게 좋을 것 같아요"

2️⃣ NANDA 분류:
   🔴 협력문제: PC: Hearing loss / PC: Depression
   🟠 실제 진단: "이명 증상과 관련된 불안 및 수면 방해"
   - 합병증 위험: 청력 손상, 이명성 우울증, 자살 위험

3️⃣ 현재 심리 지원:
   - "그 소리로 인한 불안을 함께 살펴봐요"
   - 음향 마스킹 (백색소음, 자연음)
   - 주의 전환 기술
   - 수면 위생 개선

4️⃣ 모니터링:
   "청력이나 기분 변화가 있으면 바로 알려주세요"
```

**🔴 어지럼증 — PC: Fall injury / 낙상 위험**

감지:
```
"세상이 돌아요", "어지러워요", "균형이 안 맞아요"
```

응답:
```
1️⃣ 🔴 의료 연결 (협력문제):
   "이비인후과나 신경과에 가서 원인을 확인하세요"

2️⃣ NANDA 분류:
   🔴 협력문제: PC: Fall injury
   🟠 실제 진단: "어지럼증과 관련된 낙상 불안 및 활동 제약"
   - 합병증 위험: 낙상, 골절, 외상

3️⃣ 현재 안전 + 심리 지원:
   - 안전: "천천히 움직이기", "지팡이 사용하기"
   - 심리: "낙상 공포를 함께 관리해요"
   - "활동을 천천히 재개하는 방법을 배워봐요"

4️⃣ 모니터링:
   "낙상이 있거나 어지러움이 악화되면 바로 알려주세요"
```

### 🔴 정신건강 기반 협력문제

**🔴 우울증 증상 — PC: Suicide attempt**

감지:
```
"2주일 이상 무력감, 흥미 상실"
"자꾸 울고 싶어요"
"밥맛이 없어요"
"잠을 못 자요"
```

응답:
```
1️⃣ 🔴 의료 상담 권유 (협력문제):
   "정신과나 정신건강의학과에 가서 전문가와 이야기 나누세요"

2️⃣ NANDA 분류:
   🔴 협력문제: PC: Suicide attempt
   🟠 실제 진단: "우울감과 관련된 무력감 및 활동 감소"
   - 합병증 위험: 자살 시도, 사회적 고립, 신체 건강 악화

3️⃣ 현재 심리 지원:
   함께 관리하기 (지금 바로 할 수 있는 것):
   - 하루 한 번 햇빛 노출
   - 가벼운 산책 (신체 활동)
   - 작은 목표 세우기 (밥 먹기, 샤워 등)
   - 신뢰할 수 있는 사람과 연결

4️⃣ 모니터링:
   "자살 생각이 들거나 무력감이 심해지면 바로 알려주세요"
```

**불안장애 증상**

감지:
```
"항상 불안해요"
"뭔가 나쁜 일이 일어날 것 같아요"
"손발이 떨려요"
"심장이 빨리 뛰어요"
```

응답:
```
1️⃣ 의료 상담 권유:
   "정신건강의학과나 정신건강복지센터에 가세요"

2️⃣ NANDA 분류:
   협력문제 (의료 영역) + 심리적 영향 (CareFlow 영역)
   - 진단: "불안으로 인한 신체 증상 및 활동 회피"
   - 합병증 위험: 공황장애, 광장공포증, 고립

3️⃣ 현재 불안 관리:
   지금 바로 할 수 있는 것:
   - 4-7-8 호흡법: 4초 들이마시기, 7초 멈추기, 8초 내쉬기
   - 5가지 감각: 보이는 것 5개, 들리는 것 4개... (현재 순간에 집중)
   - 신체 스캔: 발가락부터 머리까지 신체 감각 느끼기
   - 불안을 "날씨"로 보기: "폭풍이 오고 있지만 지나갈 거야"

4️⃣ 모니터링:
   "불안이 증가하거나 피하는 활동이 많아지면 알려주세요"
```

**공황장애 증상**

감지:
```
"갑자기 심장이 철렁하면서 죽을 것 같아요"
"손발이 떨리고 차가워요"
"질식할 것 같아요"
"조절할 수 없는 두려움"
```

응답:
```
1️⃣ 즉시 안정화 (발작 중):
   "지금 당신은 죽지 않을 거예요. 이건 공황 발작입니다"
   "함께 천천히 숨을 쉬어 봅시다"
   (4초 들이마시기 - 4초 멈추기 - 4초 내쉬기)

2️⃣ 의료 확인:
   "공황 발작은 의료 확인이 중요해요"
   "정신건강의학과에 가세요"

3️⃣ 발작 후 심리 지원:
   - 공황이 위험하지 않다는 것을 반복해서 상기
   - 불안이 높은 상황부터 천천히 접근하기
   - 피했던 활동 재개하기

4️⃣ 모니터링:
   "발작이 반복되거나 광장공포증이 생기면 알려주세요"
```

## 🛠️ 기술 구현

### nursingLogic.ts의 NANDA 유형 분류

```typescript
// assessMessage() 함수에서:
const nandaType = assessNANDAType(message)
// 'wellness' | 'risk' | 'actual' | 'collaborative'

if (crisisLevel === 'critical') {
  return getCrisisResponse('1393')
} else if (nandaType === 'collaborative') {
  // 협력문제: 의료 연결 + 심리 지원
  return {
    medical: getMedicalReferral(assessment.domain),
    psychological: buildSupportResponse(assessment)
  }
} else {
  // 나머지: NANDA 6단계 응답
  return buildNANDAResponse(assessment)
}
```
